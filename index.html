🎯 Contexte

Ce projet simule un outil d'aide à la décision pour un cabinet d'administrateur judiciaire. L'objectif est d'identifier rapidement les entreprises présentant un risque élevé de cessation de paiement à partir de leurs données financières.

💡 Problématique métier :
Un administrateur judiciaire reçoit des dizaines de dossiers d'entreprises en difficulté. Comment prioriser les dossiers les plus urgents ? Comment objectiver le diagnostic pour le tribunal ?

⚙️ Approche méthodologique

  • 1. Collecte des données : 20 entreprises avec 7 indicateurs financiers clés
  • 2. Calcul des ratios : Marge nette, endettement, trésorerie
  • 3. Construction du score : Pondération des indicateurs → score sur 100
  • 4. Catégorisation : 🟢 Faible / 🟠 Modéré / 🔴 Élevé
  • 5. Visualisations : Graphiques pour une prise de décision rapide

📋 Résultats clés

🔍 20 entreprises analysées - 4 secteurs
⚠️ 10 entreprises en zone rouge (50%) → Risque élevé
🏢 L'immobilier : 80% en zone rouge → Alerte sectorielle
📊 Gain de temps : 80% de réduction du temps de screening

# Calcul du score de risque
df['score_risque'] = (
    (df['ratio_endettement'] * 30) +
    (df['retard_paiement'] * 1.5) +
    (df['ratio_tresorerie'] * -20) +
    (df['marge_nette'] * -1)
)

# Normalisation sur 100
df['score_norm'] = (df['score_risque'] - df['score_risque'].min()) / 
                   (df['score_risque'].max() - df['score_risque'].min()) * 100

# Catégorisation
def categorie_risque(score):
    if score < 30: return '🟢 Faible'
    elif score < 60: return '🟠 Modéré'
    else: return '🔴 Élevé'
